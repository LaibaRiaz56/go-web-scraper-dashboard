# Go Web Scraper Dashboard

##  Overview
This project is a full-stack web application that scrapes live data from a public website, exposes it via an API, and displays it in a modern dashboard.

## ⚙️ Features
- Web scraping using Go (Colly)
- Automated CRON job (runs every 10 seconds)
- REST API built with Gin
- Modern frontend using Next.js + Tailwind CSS
- Auto-updating dashboard

## 🛠 Tech Stack
- Go (Golang)
- Gin (API)
- Colly (Scraping)
- Node-Cron equivalent (Go Cron)
- Next.js (Frontend)
- Tailwind CSS

## 🚀 How to Run

### Backend
cd backend-go  
go run .

### Frontend
cd frontend  
npm run dev
